# Dashboard Feature

Main application dashboard with project selector and navigation.

## Overview

Landing page after login displaying project selector, recent batches, quick actions, and navigation to core features.

## User Stories

- As a user, I want to see my recent activity on login
- As a user, I want quick access to create new templates or upload batches
- As a user, I want to switch projects from the dashboard

## Key Workflows

### 1. Dashboard Load
1. User logs in or navigates to dashboard
2. Current project loaded from API
3. Recent batches fetched and displayed
4. Quick action buttons shown
5. Project selector populated

## Dependencies

- **Depends on:** simple-projects, batch-view
